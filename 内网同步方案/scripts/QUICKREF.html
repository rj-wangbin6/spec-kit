⚡ 快速命令

1️⃣ 安装 CLI (一次性)

uv tool install specify-cli \
  --find-links http://172.16.37.100:9999/packages/spec-kit \
  --find-links http://172.16.37.100:9999/packages/dependencies \
  --no-index

2️⃣ 初始化项目 (每个新项目)

✅ 正确 - 完全离线 
specify init my-project --ai copilot --offline
❌ 错误 - 会尝试访问GitHub 
specify init my-project --ai copilot

3️⃣ 验证安装

specify check

⚠️ 常见错误

错误1: "Failed to download template"

原因: 忘记添加 --offline

解决: 添加 --offline 参数

错误2: "No solution found"

原因: 依赖包缺失

解决: 联系管理员或检查网络
