рҹ“Ұ жөҸи§ҲеҸҜз”ЁеҢ…

йҖҡиҝҮжөҸи§ҲеҷЁи®ҝй—®д»ҘдёӢең°еқҖжҹҘзңӢжүҖжңүеҸҜз”ЁеҢ…:

  • йЎ№зӣ®еҢ…: http://172.16.37.100:9999/packages/spec-kit/
  • дҫқиө–еҢ…: http://172.16.37.100:9999/packages/dependencies/

рҹӣ пёҸ еёёи§Ғй—®йўҳ

Q1: дёәд»Җд№ҲйңҖиҰҒ --offline еҸӮж•°пјҹ

A: еҪ“еүҚзүҲжң¬пјҲv0.0.70пјүй»ҳи®Өдјҡд»ҺGitHubдёӢиҪҪжЁЎжқҝж–Үд»¶гҖӮдҪҝз”Ё --offline еҸӮж•°еҸҜејәеҲ¶дҪҝз”ЁCLIеҶ…зҪ®иө„жәҗпјҢзЎ®дҝқе®Ңе…ЁзҰ»зәҝиҝҗиЎҢгҖӮд»Һv0.6.0ејҖе§ӢпјҢзҰ»зәҝжЁЎејҸе°ҶжҲҗдёәй»ҳи®ӨиЎҢдёәгҖӮ

Q2: еҲқе§ӢеҢ–йЎ№зӣ®ж—¶жҸҗзӨә"Failed to download"пјҹ

A: иҜҙжҳҺеҝҳи®°ж·»еҠ  --offline еҸӮж•°гҖӮиҜ·дҪҝз”Ёпјҡ

specify init my-project --ai copilot --offline

Q3: е®үиЈ…ж—¶жҸҗзӨә"No solution found"пјҹ

A: еҸҜиғҪжҳҜдҫқиө–еҢ…дёҚе®Ңж•ҙ,иҜ·иҒ”зі»з®ЎзҗҶе‘ҳжӣҙж–°жңҚеҠЎеҷЁеҢ…гҖӮ

Q4: Windowsе®үиЈ…ж—¶жҸҗзӨә"зі»з»ҹж— жі•жү“ејҖжҢҮе®ҡзҡ„и®ҫеӨҮжҲ–ж–Үд»¶"пјҹ

A: иҝҷжҳҜWindowsзҺҜеўғдёӢзҡ„дёҙж—¶ж–Үд»¶жқғйҷҗй—®йўҳпјҢе°қиҜ•д»ҘдёӢи§ЈеҶіж–№жЎҲпјҡ

ж–№жі•1: жё…зҗҶuvзј“еӯҳпјҲжҺЁиҚҗпјү

uv cache clean
uv tool install specify-cli --find-links http://172.16.37.100:9999/packages/spec-kit --find-links http://172.16.37.100:9999/packages/dependencies --no-index

ж–№жі•2: д»Ҙз®ЎзҗҶе‘ҳжқғйҷҗиҝҗиЎҢPowerShell

еҸій”®PowerShellеӣҫж Ү вҶ’ "д»Ҙз®ЎзҗҶе‘ҳиә«д»ҪиҝҗиЎҢ"пјҢз„¶еҗҺйҮҚж–°жү§иЎҢе®үиЈ…е‘Ҫд»ӨгҖӮ

ж–№жі•3: жЈҖжҹҘжқҖжҜ’иҪҜд»¶

дёҙж—¶зҰҒз”ЁWindows DefenderжҲ–е…¶д»–жқҖжҜ’иҪҜд»¶пјҢе®ҢжҲҗе®үиЈ…еҗҺеҶҚеҗҜз”ЁгҖӮ

Q5: WindowsеҲқе§ӢеҢ–йЎ№зӣ®жҸҗзӨә"pwsh not found"пјҹ

A: дҪҝз”Ё --offline еҸӮж•°еҲқе§ӢеҢ–йЎ№зӣ®ж—¶пјҢйңҖиҰҒе®үиЈ…PowerShell 7+пјҲpwshпјүпјҢWindowsеҶ…зҪ®зҡ„PowerShell 5.xдёҚж”ҜжҢҒгҖӮ

и§ЈеҶіж–№жЎҲ1: е®үиЈ…PowerShell 7+пјҲжҺЁиҚҗпјү

  1. и®ҝй—® https://aka.ms/powershell
  2. дёӢиҪҪе№¶е®үиЈ…PowerShell 7+
  3. йҮҚж–°жү“ејҖз»Ҳз«ҜпјҢдҪҝз”Ё pwsh е‘Ҫд»Өиҝӣе…ҘPowerShell 7+
  4. жү§иЎҢ specify init . --ai copilot --offline

и§ЈеҶіж–№жЎҲ2: еҺ»жҺү --offline еҸӮж•°пјҲйңҖиҰҒеӨ–зҪ‘и®ҝй—®пјү

specify init . --ai copilot

жіЁж„ҸпјҡеҺ»жҺү --offline еҗҺдјҡе°қиҜ•д»ҺGitHubдёӢиҪҪжЁЎжқҝпјҢйңҖиҰҒиғҪи®ҝй—®GitHubгҖӮеҰӮжһңеңЁеҶ…зҪ‘зҺҜеўғпјҢе»әи®®е®үиЈ…PowerShell 7+дҪҝз”ЁзҰ»зәҝжЁЎејҸгҖӮ

Q6: еҰӮдҪ•еҚёиҪҪ Specify CLIпјҹ

A: дҪҝз”Ёд»ҘдёӢе‘Ҫд»Ө:

uv tool uninstall specify-cli

Q7: жңҚеҠЎеҷЁеҢ…еӨҡд№…жӣҙж–°дёҖж¬Ўпјҹ

A: жңҚеҠЎеҷЁжҜҸеӨ©еҮҢжҷЁ2зӮ№иҮӘеҠЁд»ҺGitHubеҗҢжӯҘжңҖж–°зүҲжң¬гҖӮ

Q8: macOS/LinuxдёҠиғҪз”Ёеҗ—пјҹ

A: е®Ңе…Ёж”ҜжҢҒпјҒ24дёӘдҫқиө–еҢ…жҳҜи·Ёе№іеҸ°йҖҡз”Ёзҡ„пјҲpy3-none-anyпјүпјҢpyyamlеҢ…е·ІдёәжүҖжңүе№іеҸ°жҸҗдҫӣдё“з”ЁзүҲжң¬гҖӮжңҚеҠЎеҷЁеҪ“еүҚж”ҜжҢҒпјҡ

  • вң… Windows (x86_64)
  • вң… Linux (x86_64)
  • вң… macOS (Intel x86_64 + Apple Silicon ARM64)
